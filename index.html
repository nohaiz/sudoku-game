Instructions

  • Objective: Fill grid so each column, row, and 3x3 subgrid has numbers 1-9.
  • Starting Grid: When selecting the difficulty, certain cells are pre-filled with numbers as clues.
  • Rules: No repeats in rows, columns, or subgrids.
  • Play: Digits 1-9 must be used without repeating in rows, columns, or subgrids, playing by filling empty cells based on given clues and rules.
  • Hints: Start with cells having fewer options.
  • Reset: Resets the game to its initial stage.
  • Undo: Undoes all previously incorrect moves made.
